## 说明

一个简单的LLM角色扮演小练习

![ui-demo.png](assets/ui-demo.png)

可以自定义角色与你进行对话，它先手

![AI-self-pair-chatting.png](assets%2FAI-self-pair-chatting.png)

可以自定义两个角色，互相进行对话。

## 运行

```shell
pip install -r requirements.txt

OPENAI_API_KEY=<your_api_key_here> python app.py
```
